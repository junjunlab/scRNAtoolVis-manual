6 markerVocalno

markerVocalno is used to plot vocalno-plot for marker genes.

6.1 examples

test <- system.file(“extdata”, “pbmc.markers.csv”, package = “scRNAtoolVis”) markers <- read.csv(test)

# plot
markerVocalno(markers = markers,
              topn = 5,
              labelCol = ggsci::pal_npg()(9))
