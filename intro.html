2 Introduction

This scRNAtoolVis package is mainly used to visualize scRNA-seq related figures to produce some elegant plots.

2.1 Installation

You can install the development version of scRNAtoolVis like so:

# install.packages("devtools")
devtools::install_github("junjunlab/scRNAtoolVis")
